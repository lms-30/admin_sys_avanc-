Cloud Computing : AWS, Microsoft AZURE, Google Cloud Plateforme

### Définition
 Le cloud Computing est un modèle qui permet d'accèder à distance, via internet, à des ressources informatiques (serveurs, stockage, application) sans gérer directement l'infrastructure.
### Objectifs: 
- définir le concept de cloud Computing
- Expliquer les différents modèles de services cloud (IaaS, PaaS, SaaS)
### Caractéristique

- Accès à la demande (On-demand self-service)
- Accès réseau large (broad network access)
- Mutualisation des ressources (ressources pooling)
- Elasticité rapide (rapide elsaticity)
- Service mesuré (mesured service)

### Modèle de service du cloud

#### Infrastructure as a Service (IaaS)
- Fournit serveur, stockage
